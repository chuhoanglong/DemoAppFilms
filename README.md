## Running the project

- `yarn` or `npm install`
- Run the following steps for your platform

### start server metro
- `yarn start` or `npm install`

### iOS

- `cd ios`
- `pod install`
- `cd ..`
- `react-native run-ios`

### Android

- `yarn run android` or `npm run android`

## Detox Test

- `yarn detox:build:iosDebug` build app test
- `yarn detox:test:iosDebug` run test
